# whatver

[![npm version](https://badge.fury.io/js/%40mstssk%2Fwhatver.svg)](https://badge.fury.io/js/%40mstssk%2Fwhatver)

What is the version of this command?

## Usage

```sh
$ npm i -g @mstssk/whatver
$ whatver git
```

OR

```sh
$ npx @mstssk/whatver git
```
